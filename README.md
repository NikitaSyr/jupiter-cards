# Portfolio with cards для Jupiter

Данное тестове задание было создано по макету Adobe XD. Нажатие на категорию (в блоке выбора или по нажатию на категорию карточки) применяется фильтр по выбранной категории. Кнопка "Load more" загружает 9 новых рандомных карточек. Карточки можно выделить нажатием на неё, и, если смотреть задание с браузера компьютера, то нажатие кнопки "Delete" удалит карточки из портфолио.
Результат работы выложен на сайт: https://NikitaSyr.github.io/jupiter-cards

#### Стек использованных технологий: 
* Проект полностью типизирован
* CSS выполнен с применением методологии БЭМ. 
* Хранение данных реализованно при помощи redux и react-redux. 

Для запуска локальной верссии приложения необходимо клонировать репозиторий, а затем в терминале выполнить следующие команды:  

npm install

затем

npm start
